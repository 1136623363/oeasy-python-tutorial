### 顺序过程
- 其实代码可以分成三块
	- 输入
	- 处理
	- 输出
- 这是一个典型的面向过程的程序
- 面向过程就是Procedure Oriented

![图片描述](https://doc.shiyanlou.com/courses/uid1190679-20210816-1629054611474)

- 接收到什么信号 - 输入 - input
- 怎么处理信号 - 处理 - process
- 最终输出什么信号 - 输出 - output
- 那建立process.py

### else
- else 的意思就是否则
- 如果没有异常except 
- 就运行else里面内容

![图片描述](https://doc.shiyanlou.com/courses/uid1190679-20210816-1629057399059)

- 这就是try的完全体
