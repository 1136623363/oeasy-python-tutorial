### 再改代码

```
import requests
from lxml import etree
response = requests.get("http://localhost/oeasy.html")
s_html = response.content
#print(s_html)
et_html = etree.HTML(s_html)
et_target = et_html.xpath("//@lang")
for element in et_target:
    print(element)
```

- 运行成功
- 找到相应的属性值
